################## Gemino Alpha v0.1 - Возможности

Gemino SmartSync - Программа для локальной синхронизации данных.

- Создание "наблюдателей" для выбранной директории, которые осуществляют фоновую синхронизацию по указанному пути
- Синхронизация происходит при таких изменениях, как:
	1. Создание\удаление файла или папки
	2. Изменение содержимого файла, как текстового, так и бинарного
	3. Переименование файла или папки
	4. Вставка новых объектов в папку
- Запись лог-файлов при выполнении различных действий (отключается в настройках)
- Автозагрузка с Windows (отключается в настройках)
- Возможность установить директорию для синхронизаций по умолчанию
- Проверка и загрузка обновлений
